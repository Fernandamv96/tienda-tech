# Apunte-pre-prueba-redes

Apunte en formato HTML con contenido de repaso para la prueba de Redes, enfocado en direccionamiento IP.

## Contenido

- `apunte_pre_prueba_redes_IP.html`: documento con los apuntes de IP.

## Demo

🔗 [apunte-pre-prueba-redes.vercel.app](https://apunte-pre-prueba-redes.vercel.app)

## Uso

Abre el archivo `.html` directamente en tu navegador, o visita el enlace de la demo en línea.
