# TechStore — Gestión de Productos

Aplicación CRUD desarrollada en ReactJS para una tienda de tecnología.

## Funcionalidades
- Agregar, editar y eliminar productos
- Campos: nombre, descripción, precio, categoría, stock e imagen
- Validaciones de datos y tamaño de imagen (máx. 2 MB)
- Contador dinámico de productos
- Diseño responsivo con CSS moderno

## Tecnologías
- ReactJS
- useState, formularios controlados, eventos

## Cómo ejecutar
npm install
npm start
